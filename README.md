# 3D Flight Simulator Prototype

TypeScript と Three.js を使用して構築された、ブラウザで動作するシンプルな 3D フライトシミュレーターのプロトタイプです。航空機のピッチ操作（機首の上下）と、それに応じた 3D 空間での前進・上昇・下降の基本物理を実装しています。

## 🎮 操作方法

プロジェクトを起動後、ブラウザ画面で以下のキーボード操作を行います。

- **`↑` (矢印キー上)**: 機首を上げる（上昇）
- **`↓` (矢印キー下)**: 機首を下げる（下降）

※三人称視点（後方追従カメラ）になっているため、機体の傾きに合わせてカメラも自動で追従します。

## 🛠️ 技術スタック

- **Language:** TypeScript
- **3D Engine:** Three.js
- **Build Tool:** Vite

## 🚀 セットアップと実行方法

環境構築と開発サーバーの起動手順です。Node.js がインストールされている必要があります。

### 1. 依存関係のインストール

プロジェクトフォルダのルートで以下のコマンドを実行し、必要なパッケージをインストールします。

```bash
npm install
2. 開発サーバーの起動
ローカルでの開発・テスト用サーバーを立ち上げます。

Bash
npm run dev
コマンド実行後、ターミナルに表示される URL（例: http://localhost:5173/）にブラウザでアクセスしてください。

3. プロダクションビルド（公開用ファイルの生成）
本番環境用に最適化されたファイルを書き出す場合は、以下のコマンドを実行します。dist フォルダに生成されます。

Bash
npm run build

🛫 今後の拡張アイデア
このプロトタイプをベースに、さらに本格的なシミュレーターへ発展させるための推奨ステップです。

ロール（左右の傾き）の追加: ArrowLeft / ArrowRight で機体を傾け、旋回できるようにする。

スロットル（推力）コントロール: スピードを自由に変えられるようにし、速度不足による「失速（ストール）」を実装する。

機体モデルのインポート: GLTFLoader を使い、赤い箱からリアルな飛行機の 3D モデル（.gltf / .glb）に差し替える。
